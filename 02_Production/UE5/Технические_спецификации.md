# Технические спецификации UE5.7

> Информация для производства — обновляется по мере разработки.

## Выбор движка: Unreal Engine 5.7

### Ключевые фичи для RPG
| Фича | Применение в Magic Awakening |
|------|------------------------------|
| **Nanite** | Сложные архитектурные структуры (замки, руины) без LOD-головной боли |
| **Lumen** | Глобальное освешение для атмосферных локаций Голубой планеты |
| **Chaos Physics** | Разрушаемость окружения в магических боях (5.7 — стабильнее) |
| **MetaHuman** | NPC если нужен реализм (решить на концепте арта) |
| **World Partition** | Галактический масштаб — переходы между планетами |
| **Niagara** | Визуальные эффекты маны и скверны |

## Blueprint vs C++

### Предварительное решение
| Система | Реализация | Обоснование |
|---------|----------|-------------|
| Core gameplay (4 фазы атаки) | C++ | Производительность, точность таймингов |
| Ability editor | C++ core + BP exposed | Сложная логика + кастомизация |
| UI | Blueprint | Быстрая итерация |
| VFX | Blueprint + Niagara | Визуальная настройка |

## Пайплайн контента

### Порядок создания (Vertical Slice подход)
1. **Greybox** локации → тестируем навигацию
2. **Placeholders** персонажей → тестируем камеру
3. **Базовые VFX** → читаемость магии
4. **Анимации** → ощущение веса
5. **Полировка** → одна локация в полном качестве

## Git vs Perforce
- **Рекомендация:** Perforce для бинарников (UE5 ассеты)
- **Альтернатива:** Git LFS если команда <5 человек

## Масштабирование
- **Начало:** Одна планета (Голубая)
- **MVP:** Один регион + полная боевка
- **Full:** Мультипланетарная система Арона
