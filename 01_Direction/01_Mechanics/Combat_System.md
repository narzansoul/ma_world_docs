# Боевая система

> Статус: Эволюционирует. Базовая концепция → уточнение.

## Концепт: Кастомизируемая магия

### Типы способностей
1. **Игровые** — предустановленные создателями
   - Получаются через прохождение
   - Изучение библиотек
   
2. **Пользовательские** — редактор способностей
   - Создание алиасов/заготовок
   - Конфигурация формы и наполнения

### 4 фазы атаки (константы)

```
┌─────────────┐    ┌──────────────────┐    ┌─────────────────┐    ┌──────────────┐
│  1. Триггер │ →  │2. Скапливание    │ →  │3. Всплеск       │ →  │4. Попадание  │
│  (клавиша)  │    │   (анимация,     │    │   (направление, │    │   (урон)     │
│             │    │    наполнение    │    │    стихия)      │    │              │
│             │    │    элементом)    │    │                 │    │              │
└─────────────┘    └──────────────────┘    └─────────────────┘    └──────────────┘
```

### Система должна позволять:
1. Управлять анимацией атаки
2. Связывать с константами (мана, кулдауны)
3. Конфигурация формы и наполнения
4. Зависимость от навыков и опыта

## Референс: Hogwarts Legacy
- **Что взять:** Визуальное исполнение заклинаний, ощущение мощи
- **Что развить:** Глубже кастомизация (не просто выбор из списка, а конструктор)
- **Что избежать:** Простое переключение между готовыми заклинаниями
